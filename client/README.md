# Courier Onboarding Wizard

A React and TypeScript multi-step onboarding wizard for prospective couriers.

## Requirements

- Node.js
- npm

## Running the project

Install dependencies:

```
npm install
```

Start the mock API and React app together:

```
npm run dev
```

This starts:

- Mock API: `http://localhost:4000`
- React app: `http://localhost:5173`

## Tests

Run tests:

```
npm test
```

## Build

Create a production build:

```
npm run build
```
